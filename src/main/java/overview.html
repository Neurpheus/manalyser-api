Aplication Programming Interface (API) for Morphological Components.

The API defines basic interfaces and classes which can be used to perform a morphological analysis of natural languages. The API defines interfaces for the following kinds of components related to the morphology of natural languages:

  • Morphological Analyser
  • Stemmer
  • Lemmatizer
  • Word Form Generator
  • Tagset

The following fragment of code presents typical application of the API:


MorphologyFactory factory = DefaultMorphologyFactory.getInstance();
Stemmer stemmer = factory.getMostAccurateStemmer(Locale.ENGLISH);
String stem = stemmer.getStemm(“testing”);
...
MorphologicalAnalyser analyser =
    factory.getAnalyser(new Locale(“pl”), PropertyConst.VENDOR, “neurpheus”);
MorphologicalAnalysisResult[] result = analyser.getAnalisisResults(“testing”);

More information about the API can be found in the descriptions of two following packages:

  • {@link org.neurpheus.nlp.morphology}
  • {@link org.neurpheus.nlp.morphology.tagset}
